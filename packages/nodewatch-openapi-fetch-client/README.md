# nodewatch-openapi-fetch-client

このプロジェクトは、NodeWatch OpenAPI を利用して NodeWatch REST クライアントを生成するためのものです。  
This project is for generating a NodeWatch client using the NodeWatch OpenAPI.

## 特徴 / Features

- OpenAPI 仕様に基づいて自動生成されたコード / Code automatically generated based on the OpenAPI specification

## OpenAPI / Swagger

- https://nodewatch.symbol.tools/api/openapi
