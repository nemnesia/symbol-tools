# 変更履歴

このプロジェクトにおけるすべての重要な変更は、このファイルに記録されます。

変更履歴のフォーマットは[変更履歴の管理](https://keepachangelog.com/en/1.0.0/)に基づいています。

## [0.2.0] - 2026/07/23

### 修正

- WebSocket の異常切断時に接続状態、UID、購読ハンドルをリセットするよう改善。
- 未接続中に `off()` した購読が、後続の接続時に復元される問題を修正。
- 同一チャネルの複数コールバックを追跡し、再接続時の復元と `off()` による解除を正しく処理するよう改善。
- 重複する再接続タイマーの登録を防止。
- `timeout` と `ssl` の既定値の型ドキュメントを実装に合わせて修正。
- `@stomp/stompjs`、`ws` の依存バージョンを更新。

### ドキュメント

- ESM 専用パッケージとしての利用要件と、`off()` の解除範囲を README に明記。
- 公開 API の JSDoc に、接続・再接続・購読解除・エラー分類の実際の挙動を追記。
