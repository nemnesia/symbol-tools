# 変更履歴

## [0.1.0] - 2025/12/29

### 追加

- Symbol ブロックチェーンのWebSocket接続をサポート。
- リアルタイムデータ取得機能（ブロック、トランザクション、アカウント情報など）。
- 柔軟なサブスクリプション管理機能。
- 自動再接続機能とサブスクリプション復元機能。
- エラーおよびクローズイベントのハンドリング機能。
