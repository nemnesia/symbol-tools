# Changelog

## [0.2.0] - 2026-07-23

### Added

- バージョン、KDF、KDF パラメータ、暗号方式を含み、これらを AAD として認証する暗号化データ形式を追加
- 厳密な RFC 4648 Base64 処理と、平文・暗号文のサイズ制限を追加
- v1 より前の `{ salt, ciphertext }` 形式を移行目的で復号する互換性を追加

### Changed

- Argon2id を非同期実装へ変更し、パスワード由来バイト列と導出鍵を使用後にゼロ埋め
- カバレッジ実行時の Argon2id テストがタイムアウトしないよう、Vitest の既定タイムアウトを 30 秒に変更
- API ドキュメントと README のセキュリティおよび移行に関する説明を更新
